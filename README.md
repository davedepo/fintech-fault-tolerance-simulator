## Overview
A visual simulation demonstrating how fault tolerance operates in payment systems.  
Built to illustrate resilience patterns in distributed financial infrastructure.

## Key Concepts Demonstrated

- Primary and backup payment routing paths  
- Automatic failover during service disruption  
- Core resilience principles in payment architecture
